# 🌳 Family Root

A beautiful and intuitive mobile application designed to help individuals record, visualize, and preserve their complex family trees across generations.

---

## 🚀 Purpose & Vision
Modern family structures are rich and deeply connected, yet traditional software often struggles to capture the full picture seamlessly. **Family Root** bridges this gap by offering a clean user interface that maps out relationships, handles historical calendars, and ensures that lineage data is easy to log and interactive to navigate.

## ✨ Core Features
* **Interactive Family Tree Mapping:** Easily add members, link relationships, and view lineage structures dynamically.
* **Dual Calendar Support:** Integrated preferred calendar options (including regional formats) to track birthdays and significant historical events accurately.
* **Detailed Profiles:** Keep track of individual member details, sibling connections, and deep generational history.
* **Offline-Ready Performance:** Built to secure and hold your family data flawlessly.

---

## 🛠️ Technical Breakdown
This application is engineered using modern, scalable web and mobile technologies:
* **Frontend Framework:** React with TypeScript (leveraging rigid types for complex relationship trees)
* **Styling Architecture:** Tailwind CSS (configured with dark mode support)
* **Build Tooling:** Vite & Capacitor Framework
